Install on Android

Recommended — real app file (APK)

  1. Tap Download Android app (APK).
  2. Open the file when download finishes.
  3. If asked, allow installs from Chrome / Files.
  4. Tap Install, then open Study Tracker.

Browser shortcut (optional)

  1. In Chrome, tap ⋮ (top right).
  2. Tap Add to Home screen or Install app if shown.
