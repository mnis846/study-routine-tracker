Install (phone or tablet only)

No laptop required. Use Chrome.

  1. Open this page (the study tracker link) in Chrome.
  2. Tap the ⋮ menu (top right).
  3. Tap Install app or Add to Home screen.
  4. Open the new icon anytime — works offline. Study data stays on this device.
